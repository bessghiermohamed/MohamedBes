# مواصفات نموذج تقييم الزملاء

## 1. المفهوم والرؤية

نموذج تقييم زملاء باللغة العربية مصمم للمدرسة العليا/تكوين المعلمين. يجمع بين الأسئلة الأكاديمية الجادة والأسئلة الممتعة/الافتراضية لخلق تجربة تفاعلية ممتعة. التصميم يعكس الدفء الأكاديمي مع لمسة عصرية.

## 2. لغة التصميم

### الاتجاه الجمالي
- تصميم عربي أصيل مع لمسات عصرية
- ألوان دافئة تعكس بيئة أكاديمية مريحة

### لوحة الألوان
- **الأساسي**: `#2D5016` (أخضر غامق أكاديمي)
- **الثانوي**: `#F5E6D3` (بيج دافئ)
- **التمييز**: `#D4A574` (ذهبي ترابي)
- **الخلفية**: `#FBF9F7` (أبيض كريمي)
- **النص**: `#1A1A1A` (أسود ناعم)
- **النص الثانوي**: `#6B6B6B` (رمادي)

### الخطوط
- العناوين: Cairo (Google Fonts) - Bold
- النص: Cairo - Regular
- الأحجام: عناوين 2.5rem، نص عادي 1rem

### الحركة
- انتقالات سلسة 0.3s ease
- تأثيرات hover خفيفة على البطاقات
- تحريك ظهور الأقسام عند التمرير

## 3. الهيكل والتخطيط

### بنية الصفحة
1. **الرأس**: شعار + عنوان النموذج
2. **قسم الترحيب**: مقدمة قصيرة
3. **القسم الأول**: الأساتذة وطرق التدريس (تقييم بالنجوم)
4. **القسم الثاني**: الجو الدراسي والعلاقات (خيارات متعددة)
5. **القسم الثالث**: البنية التحتية والإمكانيات (تقييم رقمي)
6. **القسم الرابع**: أسئلة ممتعة/افتراضية
7. **قسم إرسال البيانات**: إدخال البريد الإلكتروني + زر الإرسال

### التجاوب
- تصميم متجاوب للموبايل والديسكتوب
- بطاقات مرنة تتكيف مع الشاشة

## 4. الميزات والتفاعلات

### أنواع الأسئلة
1. **تقييم بالنجوم (1-5)**: للأساتذة وطرق التدريس
2. **خيارات متعددة**: للعلاقات والجو الدراسي
3. **مقياس رقمي (1-10)**: للبنية التحتية
4. **نص حر**: للآراء والملاحظات
5. **أسئلة اختيارية ممتعة**: dilemas مرحة

### التفاعلات
- تحديد النجوم بالنقر
- إرسال البيانات إلى Formspree (بريد إلكتروني)
- رسالة نجاح بعد الإرسال
- التحقق من صحة البريد الإلكتروني

## 5. المكونات

### بطاقة السؤال
- خلفية بيضاء مع ظل خفيف
- حواف مستديرة 12px
- padding داخلي 24px

### مجموعة النجوم
- 5 نجوم قابلة للتحديد
- تأثير hover ذهبي
- حالة محددة ثابتة

### زر الإرسال
- خلفية خضراء داكنة
- نص أبيض
- hover: تكبير خفيف
